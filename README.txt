Zadify

Capstone Project for Zach Durtschi

The project plan is included in the repository. The final product will be a gamification Android application.

This project is written in Visual C#. It also uses Mono for Android. Unit Testing is done using MonoDroid and Visual Studio 2010's native unit testing. Logging is done through Android's own logging system. Commits are made with Github for Windows.

Progress and bug tracking can be found at https://trello.com/b/tigHJ6iz, or in Schedule.txt
