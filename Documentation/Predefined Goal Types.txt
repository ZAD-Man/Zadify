-Diet
	Lose # [weight] by [date]
	Gain # [weight] by [date]
-Finance
	Save $ by [date]
	Save $ every [timespan]
	Pay off loan ($) by [date]
	Pay off $ of loan every [date]
-Reading
	Read # [thing] per [timespan]
	Read # [thing] by [date]
-Writing
	Write # [thing] per [timespan]
	Write # [thing] by [date]
